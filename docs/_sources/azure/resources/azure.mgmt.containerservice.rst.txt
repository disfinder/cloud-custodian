.. _azure.aks:

azure.aks
=========

Azure Kubernetes Service Resource

:example:

Returns all aks clusters that did not provision successfully

.. code-block:: yaml

    policies:
      - name: broken-aks
        resource: azure.aks
        filters:
          - type: value
            key: properties.provisioningState
            op: not-equal
            value_type: normalize
            value: succeeded
